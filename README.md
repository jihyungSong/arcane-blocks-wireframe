# Arcane Blocks - UI Wireframe

✨ **Arcane Blocks** 모바일 퍼즐 게임의 UI 와이어프레임입니다.

## 🎮 Live Preview

👉 **[와이어프레임 보기](https://jihyungSong.github.io/arcane-blocks-wireframe/)**

모바일에서 확인하면 더 실감납니다!

## 📱 화면 구성

| 화면 | 설명 |
|------|------|
| 🏠 홈 | 메인 화면, 게임 시작, 최고 점수 |
| 🎮 게임 | 10x10 그리드, 블록 배치, 스킬 버튼 |
| 💀 게임오버 | 점수 표시, 광고 보고 이어하기 |
| 🧙 캐릭터 | 캐릭터 선택/수집/레벨업 |
| ⚙️ 설정 | 사운드, 진동 등 옵션 |

## 🎨 디자인 컨셉

- **테마:** 마법사 / 비전 마법 (Arcane Magic)
- **컬러:** 보라/남색 + 금색 포인트
- **느낌:** 신비롭고 마법스러운 분위기

## 🛠 기술 스택

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript

## 📋 게임 기획

- **게임 타입:** Block Blast 스타일 퍼즐
- **특징:** 캐릭터 수집/육성 시스템
- **수익화:** 배너 광고 + 전면 광고
